Начинки от Golden Cake

Уважаемые заказчики, просим вас при наличии аллергии на какие-либо продукты сообщить менеджеру.

1. Изысканный Матча

Сливочный *Женуаз - Genoise* — классический белый бисквит (яйца, сахар, мука) с добавлением сливочного масла.

*Шибуст-Chiboust* –заварной крем с добавлением мягкой меренги и взбитых сливок.

Прослойка из клубничного конфи и японского чая «Матча».

2. Роскошный Тонко

*Джоконда – Joconde* - классический белый бисквит (яйца, сахар, мука) с добавлением орехов и сливочного масла

Тонко - *Дипломат- Diplomat* - заварной крем с бобами «Тонко» и со взбитыми сливками.

Прослойка из соленой карамели и арахиса.

3. Нежнейший йогуртовый

*Женуаз - Genoise* — классический белый бисквит (яйца, сахар, мука) с добавлением сливочного масла.

Йогуртовый *Cream Cheese* - воздушный йогуртово-сырный крем.

Прослойка из натуральных ассорти цельных ягод (малина, клубника, черная смородина, черника).

4. Современная «Черная смородина»

*Шифон – Chiffon* - классический белый бисквит (яйца, сахар, мука) с добавлением сливочного масла, приготовленный по специальной авторской технологии.

«Намелака» - нежный ягодный крем на основе ганаша из чёрной смородины со взбитыми сливками.

Прослойка из крема «пломбир» и белого ганаша с ванилью.

5. Лучший из лучших сметанник «Клубника»

Сметанный *Женуаз – Genoise* - классический белый бисквит (яйца, сахар, мука) с добавлением сметаны.

Восхитительный крем из густой натуральной сметаны и клубничным кули.

Добавляются кусочки клубники.

6. Авторский ягодный «Облепиха»

Сметанный *Женуаз - Genoise* - классический белый бисквит (яйца, сахар, мука) с добавлением сметаны.

«Намелака» - нежный ягодный крем на основе ганаша из облепихи со взбитыми сливками.

Прослойка из шоколадного *Шибуст-Chiboust* –заварной крем с добавлением мягкой меренги, взбитых сливок и молочного шоколада.

7. Экстравагантный «Цитрус»

*Ангел – Angel* - белый бисквит, приготовленный только из яичных белков, с добавлением цедры лимона.

*Дипломат- Diplomat*- цитрусовый заварной крем из лайма и лимона со взбитыми сливками.

Прослойка из «крамбла» - крошка из орехов и белого шоколада.

8. Маковая экзотика «Мак и маракуя»

Маковая *Джоконда-Joconde* - классический белый бисквит (яйца, сахар, мука) с добавлением мака.

Сырный *Cream Cheese* - воздушный сырный крем приготовленный по уникальной технологии.

Пропитка сиропом из маракуйи.

Прослойка из кусочков мармелада маракуйи.

9. Незабываемый чизкейк «Манго»

*Sable* - нежные песочные коржи (яйца, сахар, мука), приготовленные по уникальной авторской технологии.

*Cream Cheese* - воздушный сырный крем с пюре из манго.

Прослойка из кусочков аппетитного персика.

10. Шоколадно-вишневый шедевр

Шоколадный *Пан де Жен- Pain de Genes* - классический шоколадный бисквит с добавлением ореховой муки, и приготовленный по эксклюзивной технологии.

Шоколадный *Дипломат-Diplomat* - заварной шоколадный крем на сливках.

Прослойка из вишневого «конфи».

11. «Орехи в шоколаде»

Шоколадно-ореховый *Дакуаз – Dacquoise* - шоколадный бисквит, приготовленный только из яичных белков, с добавлением орехов.

Шоколадный *Дипломат-Diplomat* - заварной крем шоколадный со взбитыми сливками.

Ореховый *Дипломат-Diplomat* - заварной крем с орехами и взбитыми сливками.

Прослойка из «крамбла» (рассыпчатая ореховая крошка).

12. «Шоколадные берега»

Шоколадный *Пан де Жен - Pain de Genes* - шоколадный бисквит с добавлением ореховой муки и яичных желтков, и приготовленный по эксклюзивной технологии.

Шоколадный микс *Дипломат-Diplomat* - заварной крем на сливках на основе темного, молочного и белого шоколада.

13. «Открытый восторг»

Медовые коржи в авторском исполнении.

Насыщенный сырно-сметанный крем с ванилью.

14. «Стоп глютен»

Шоколадный *Ангел – Angel* - шоколадный бисквит, приготовленный только из яичных белков на кукурузной муке.

*Англез-Anglaise* - нежный крем на основе молочного шоколада.

Прослойка из апельсинового «конфи».

15. «Стоп лактоза»

Кокосовый *Дакуаз – Dacquoise* - белый бисквит, приготовленный только из яичных белков, с добавлением кокосовой стружки.

Кокосовый *Бланманже – Blancmange* - заварной крем на кокосовых сливках с кокосовой стружкой. Приготовлен по специально разработанной технологии без использования молочных продуктов.

Прослойка из ананасового компоте – желейный крем на основе ананасового пюре.

16. «STOP AGGS»

Особый *Шифон – Chiffon* - белый бисквит с добавлением сливочного масла, приготовленный по специальной авторской технологии без использования яиц.

Сырный *Cream Cheese* - воздушный сырный крем, приготовленный по уникальной технологии.

Прослойка из ароматных карамелизированных бананов.
